# Math Blitz

A console-based arithmetic quiz game written in Java.

## Features

* Addition quizzes
* Subtraction quizzes
* Multiplication quizzes
* Score tracking
* Randomized questions

## How to Run

Compile:

```bash
javac Main.java
```

Run:

```bash
java Main
```

## Future Features

* Division mode
* Difficulty levels
* Timed challenges
* High scores
